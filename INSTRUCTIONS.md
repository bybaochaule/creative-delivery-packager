## How To Use The Skill

Ask for a delivery system, export plan, package audit, naming convention, ZIP workflow, client handoff, or marketplace-ready folder structure.

Example prompts:

```text
Create a delivery package for an Etsy PNG clipart bundle with 50 transparent files.
```

```text
Audit my product ZIP structure and tell me what is missing.
```

```text
Create a client handoff folder for a logo design project.
```

```text
Create preview image rules and thumbnail rules for a Creative Market template pack.
```

## Minimum Input To Provide

For best results, provide:

- Product name
- Product type
- Marketplace or client type
- File formats needed
- Source files included or excluded
- Number of assets
- Software used
- License terms
- Whether previews or thumbnails are needed

## Quality Standard

A good final package should be:

- Easy to understand
- Clearly named
- Separated by file type and use case
- Free of drafts and private files
- Accurate to the listing or client agreement
- Tested after ZIP extraction
- Supported by README and license files
