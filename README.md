# Creative Delivery Packager

Creative Delivery Packager is a workflow skill for organizing, exporting, packaging, auditing, and documenting professional creative file deliveries.

Use it for digital products, marketplace listings, client handoffs, branding kits, photography packages, video deliverables, prompt products, templates, previews, thumbnails, ZIP packages, and final delivery checklists.

## What It Creates

- Clean folder structures
- File naming conventions
- Export format plans
- Preview and thumbnail rules
- README and license placement
- ZIP packaging workflows
- Quality-control checklists
- Client handoff notes
- Marketplace upload preparation
- Audit reports for messy file packages

## Best For

- Etsy digital products
- Creative Market products
- Gumroad downloads
- PromptBase products
- Canva template packs
- PNG clipart bundles
- SVG cut file bundles
- Printable PDF packs
- Branding kits
- Logo handoffs
- Photography deliveries
- Video editing deliveries
- Agency client packages

## Safety Principles

This skill should not be used to package stolen assets, unlicensed fonts, copyrighted characters, fake licenses, misleading previews, private client files, hidden scripts, malware, or deceptive ZIP contents.

Users are responsible for verifying licensing, copyright, trademark, marketplace rules, client permissions, and software compatibility before publishing or delivering files.

## Suggested Repository Structure

```text
creative-delivery-packager/
  README.md
  SKILL.md
  INSTRUCTIONS.md
```

## Disclaimer

This workflow supports creative organization, packaging, export planning, and quality control. It does not guarantee marketplace approval, legal compliance, copyright clearance, trademark clearance, software compatibility, client acceptance, or commercial results.
